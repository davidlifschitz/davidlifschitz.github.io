Install one trusted runner shortcut, then generate mobile workflow payloads.

This web app avoids unsigned shortcut-file imports by sending users through the iCloud Shortcut install flow. Then it can use your own LLM API key to turn plain English into a structured runner payload.

Phone setup

  1. Open this page on your iPhone.
  2. Tap Open iCloud Runner Shortcut.
  3. Add the shortcut in Apple Shortcuts.
  4. Use a session-only API key for the safest BYOK flow, or save an encrypted key on this device.
  5. Generate a runner payload and copy it into the installed shortcut flow.

LLM settings

Bring your own API key. Session-only usage is safest. Saving is optional and stores only an encrypted blob in this browser. The key is still decrypted in browser memory while a provider request is running.
